## 🌿 About Releaf
Releaf is an eco-task web app where users unlock topics, complete green challenges, earn points, and track their environmental impact.  
This repository is a **preview** only. The full source code is private.  

Coming soon as open-source — Releaf aims to become the go-to eco-habit tracker for everyone.  If you need now be part of Releaf contact: **releafecobd@gmail.com / anik.byteinfo11@gmail.com**

---

## 🚀 Features

### 👥 User Features
- Secure user registration & login
- Progressive topic unlocking
- Task completion tracking
- Profile updates with points-based gamification

### 🛠 Admin Features
- Manage eco-topics (create, edit, delete)
- Manage tasks for each topic
- View & approve users
- Approve / delete tasks

## 👤 User Panel Overview  

The **User Panel** allows each member to track their eco-journey, manage tasks, and interact with the community.  

---

### 📌 Navigation Menu  

![Dashboard](https://img.shields.io/badge/Dashboard-1E90FF?style=for-the-badge&logo=speedtest&logoColor=white)  
- View overall progress at a glance.  
- Quick shortcuts for easy navigation.  

![Tasks](https://img.shields.io/badge/Tasks-228B22?style=for-the-badge&logo=leaflet&logoColor=white)  
- Explore tasks under 🌿 **Greenverse** and 🎉 **Funlab**.  
- To complete a task, users must **submit proof** (photo, video, audio, or text).  
- Submission goes **pending → sent to Admin Task Review**.  
- Once approved ✅, user XP increases.  

![Achievements](https://img.shields.io/badge/Achievements-FFD700?style=for-the-badge&logo=googlekeep&logoColor=black)  
- Unlock badges 🏅 based on milestones (e.g., *Eco Warrior*, *Green Hero*).  

![Notices](https://img.shields.io/badge/Notices-FF4500?style=for-the-badge&logo=pinboard&logoColor=white)  
- View all announcements shared by Admin.  

![Groups](https://img.shields.io/badge/Groups-FF69B4?style=for-the-badge&logo=wechat&logoColor=white)  
- Join groups created by Admin.  
- Group chat feature (💬 upcoming) → users can chat inside groups.  
- ⚠️ Admin cannot view these chats (privacy-first).  

![Messages](https://img.shields.io/badge/Messages-1E90FF?style=for-the-badge&logo=maildotru&logoColor=white)  
- Read direct messages sent from Admin.  

![Profile](https://img.shields.io/badge/Profile-8A2BE2?style=for-the-badge&logo=iconify&logoColor=white)  
- Upload a profile picture 🖼  
- Update username & email (max 3 times).  
- **Delete Account** option (requires password confirmation).  
  - ⚠️ Once deleted, account cannot be restored.  

---

✨ The User Panel ensures a smooth eco-tracking experience, motivating users through tasks, XP, and achievements.

## 🗺️Future Vision

We are constantly working to make Releaf more impactful. Here are some exciting features planned for the future:

-   **💬 Group Chat Functionality:** To foster a stronger community, we will implement a real-time chat feature within groups, allowing users to connect and collaborate on their eco-journey.

-   **🛍️ EcoStore:** An integrated marketplace where users can redeem their hard-earned XP to purchase sustainable products or get exclusive discounts from eco-friendly brands.

-   **🌐 EcoVast Hub:** A centralized platform to bring all eco-related organizations under one roof. This will allow users to easily discover and connect with local and global environmental initiatives, events, and volunteer opportunities directly from the Releaf app.
